Importing Gcode Files
=====================

Through sequences **GcodeFileInkScape** and **GcodeFile3DSlicer**, locally stored Gcode file generated from Inkscape or 3DSlicer software can be imported into Polychemprint3, which exapnds the amount usebale sequences to not just pre-programed simple patterns, but also convoluted structures created by users
